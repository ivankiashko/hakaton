# Планировщик ремонта - Презентация для хакатона

## Проблема

**8 из 10 собственников квартир не знают:**
- Какие работы можно делать без согласования
- Что демонтаж несущей стены может обрушить дом
- Что незаконная перепланировка грозит штрафами и судом

**Результат:**
- 40% перепланировок делаются незаконно
- Миллиарды рублей штрафов ежегодно
- Тысячи судебных дел о принудительной продаже квартир
- Стресс и головная боль для миллионов людей

## Наше решение

**"Планировщик ремонта"** — первый в России умный сервис для планирования перепланировки:

### Что делает наш продукт?

1. **Интерактивный редактор** — рисуйте план своей квартиры
2. **AI-анализатор** — мгновенная проверка по 12+ категориям законодательства
3. **Система предупреждений** — конкретные рекомендации и риски
4. **Оценка сроков и стоимости** — от 0 до 6 месяцев, от 0 до 150,000₽

### Ключевые преимущества

* **Мгновенный анализ** — результат за 1 секунду
* **Визуализация** — видите план перед ремонтом
* **Юридическая база** — ЖК РФ, СНиП, СанПиН
* **Простота** — понятно даже без юридического образования
* **Экономия** — избегайте штрафов и переделок

## Уникальность

### Что есть на рынке сейчас?

| Решение | Проблемы |
|---------|----------|
| Консультации с юристами | Дорого (от 5000₽), медленно (несколько дней) |
| БТИ | Очереди, бюрократия, неудобно |
| Форумы и статьи | Противоречивая информация, не персонализировано |
| Проектные организации | Очень дорого (от 50,000₽), долго (недели) |

### Наше решение

- **Мгновенно** — 1 секунда вместо нескольких дней
- **Доступно** — бесплатно для базового анализа
- **Наглядно** — визуальный редактор вместо текста
- **Умно** — автоматический анализ вместо ручного
- **Удобно** — онлайн, без очередей и бюрократии

## Рынок и монетизация

### Целевая аудитория

**Первичная (B2C):**
- Собственники квартир, планирующие ремонт
- 10+ млн квартир ремонтируются в России ежегодно
- Средний чек: 500-2000₽ за расширенный анализ

**Вторичная (B2B):**
- Застройщики (консультация для новых собственников)
- Ремонтные компании (проверка планов клиентов)
- Проектные организации (первичная проверка)

### Модель монетизации

1. **Freemium** — базовый анализ бесплатно
2. **Premium** — 500₽/план
   - Детальный анализ
   - 3D визуализация
   - Генерация документов
3. **Expert** — 2000₽/консультация
   - Онлайн консультация с экспертом
   - Помощь в согласовании
4. **B2B** — от 50,000₽/месяц
   - API интеграция
   - White label решение

### Прогноз выручки

**Год 1:**
- 100,000 пользователей
- 5% конверсия в Premium
- 1% конверсия в Expert
- **Выручка: ~7-10 млн ₽**

**Год 3:**
- 1,000,000 пользователей
- 10% конверсия в Premium
- 3% конверсия в Expert
- 10 B2B клиентов
- **Выручка: ~100+ млн ₽**

## Технологии

### Frontend
- React 18 + TypeScript
- Vite для быстрой разработки
- TailwindCSS для UI
- Konva.js для интерактивной графики

### Backend
- FastAPI (Python) — современный и быстрый
- Pydantic для валидации
- JSON база знаний (легко расширяемая)

### Почему именно эти технологии?

- **Скорость разработки** — MVP за 48 часов
- **Производительность** — мгновенный анализ
- **Масштабируемость** — легко добавлять новые правила
- **Экономичность** — open source стек
