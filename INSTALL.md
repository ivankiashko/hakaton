# Инструкция по установке

## Для пользователей

### Windows

1. **Скачайте установщик**
   - Перейдите в раздел [Releases](../../releases)
   - Скачайте файл `Планировщик Ремонта-Setup-1.0.0.exe`

2. **Установите Python 3.11+** (если ещё не установлен)
   - Скачайте с https://python.org/downloads/
   - При установке обязательно отметьте "Add Python to PATH"

3. **Запустите установщик**
   - Дважды кликните по скачанному `.exe` файлу
   - Следуйте инструкциям мастера установки
   - Выберите папку для установки (по умолчанию: `C:\Program Files\Планировщик Ремонта\`)

4. **Запустите приложение**
   - Найдите ярлык на рабочем столе
   - Или запустите из меню Пуск

### macOS

1. **Скачайте установщик**
   - Перейдите в раздел [Releases](../../releases)
   - Скачайте подходящий DMG файл:
     - `Планировщик Ремонта-1.0.0-x64.dmg` для Intel Mac
     - `Планировщик Ремонта-1.0.0-arm64.dmg` для Apple Silicon (M1/M2/M3)

2. **Установите Python 3.11+** (если ещё не установлен)
   - Вариант 1 (Homebrew):
     ```bash
     brew install python@3.11
     ```
   - Вариант 2: Скачайте с https://python.org/downloads/

3. **Установите приложение**
   - Откройте скачанный `.dmg` файл
   - Перетащите приложение в папку Applications

4. **Запустите приложение**
   - Откройте папку Applications
   - Найдите "Планировщик Ремонта"
   - Дважды кликните для запуска

   **При первом запуске:**
   - Может появиться предупреждение безопасности
   - Откройте Системные настройки → Безопасность
   - Нажмите "Все равно открыть"

## Требования

- **Python 3.11 или выше** (обязательно)
- **Windows 10/11** или **macOS 10.13+**
- **200 MB** свободного места на диске
- **8 GB** оперативной памяти (рекомендуется)

## Что НЕ требуется

- ✅ Node.js (встроен в приложение)
- ✅ Docker (не требуется)
- ✅ Git (не требуется)
- ✅ Дополнительные библиотеки

## Устранение проблем

### Windows

**Проблема:** "Python не найден"
- **Решение:** Установите Python 3.11+ и убедитесь, что отметили "Add Python to PATH" при установке

**Проблема:** "Windows защитила ваш компьютер"
- **Решение:** Нажмите "Подробнее" → "Выполнить в любом случае"

### macOS

**Проблема:** "Не удается открыть, так как это приложение получено не из App Store"
- **Решение:**
  1. Откройте Системные настройки → Безопасность
  2. Нажмите "Все равно открыть"

**Проблема:** "Python не найден"
- **Решение:** Установите Python через Homebrew: `brew install python@3.11`

## Обновление

Для обновления приложения:

1. Скачайте новую версию из [Releases](../../releases)
2. Удалите старую версию (опционально)
3. Установите новую версию

Ваши данные и настройки сохранятся.

## Удаление

### Windows

1. Откройте Панель управления → Программы и компоненты
2. Найдите "Планировщик Ремонта"
3. Нажмите "Удалить"

### macOS

1. Откройте папку Applications
2. Найдите "Планировщик Ремонта"
3. Перетащите в Корзину
4. Очистите Корзину
